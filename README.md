🎯 Online Quiz App

A Full Stack Online Quiz Application built using React, Flask, and MongoDB. The application allows users to log in, take quizzes, submit answers, and view their scores instantly through a clean and responsive interface.



## 🚀 Features

- 🔐 User Login
- 📝 Interactive Quiz Interface
- ⏱️ Timer-based Quiz
- ✅ Instant Score Calculation
- 📊 Result Display
- 📱 Responsive UI
- 🔗 REST API Integration
- 💾 MongoDB Database Support
